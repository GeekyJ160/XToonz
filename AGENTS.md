The goal of this project is to build an application similar to **Rizz GPT**, focusing on enhancing users' conversational skills through AI-driven suggestions, personalization, and gamification.

## General Guidelines

1.  **Follow the Plan:** Adhere to the established plan steps. If changes are needed, update the plan using the `set_plan` tool and communicate significant deviations.
2.  **Code Clarity:** Write clean, well-commented code.
3.  **Testing:** Implement tests for new features and bug fixes. Strive for test-driven development where practical.
4.  **Commit Messages:** Use conventional commit messages: a short subject line (max 50 chars), a blank line, and a detailed body if necessary.
5.  **Tool Usage:** Utilize the provided tools effectively. Do not ask the user to perform tasks you can do (e.g., installing dependencies, running commands).
6.  **Ethical AI:** Be mindful of ethical considerations, data privacy, and bias mitigation as outlined in the project plan.
